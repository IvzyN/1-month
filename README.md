## Getting Started

### Prerequisites
To run this project, you will need to have the following installed:
- Node.js
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IvzyN/1-month.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 1-month
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

## Deployment on Vercel
To deploy this application on Vercel:
1. Sign in or sign up on [Vercel](https://vercel.com).
2. Click on the 'New Project' button.
3. Import your GitHub repository.
4. Configure any necessary environment variables.
5. Click 'Deploy' and wait for the process to finish.
6. Visit the provided URL to view your deployed application.
